# Vinyl Shop – GitHub Pages Version

Dieses Projekt ist so vorbereitet, dass es direkt mit GitHub Pages funktioniert.

## Struktur

- `index.html` – Startseite
- `shop.html` – Shop-Seite
- `assets/` – Bilder, CSS und JavaScript

## GitHub Pages einrichten

1. Dieses Repo nach GitHub hochladen (Inhalt dieses ZIPs direkt ins Root des Repos legen).
2. In den **Settings** deines Repos:
   - Links im Menü **Pages** auswählen
   - Unter **Source** den Branch (z. B. `main`) auswählen und den Root-Ordner `/` einstellen
3. Speichern – nach kurzer Zeit ist die Seite online.

Danach sollten alle Bilder korrekt unter `assets/img/...` geladen werden.
